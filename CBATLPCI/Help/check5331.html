SQL Server on Domain Controller

Check Description

This check determines whether Microsoft® SQL Server™ is running on a system that is a domain controller.

We recommend that you do not run SQL Server on a domain controller. Domain controllers contain sensitive data, such as user account information, and they should not be used in another role. If you run a SQL Server database on a domain controller, you increase the complexity involved in securing the server and protecting against attack.
