Operating System Version

Issue

If you are using the Microsoft® Windows NT® operating system, your operations might not be as secure as systems running Windows® Server 2008, Windows® Server 2003, Windows Vista, Windows XP, or Windows 2000, which provide a higher level of security.

Solution

Upgrade your operating system to Windows Server 2008, Windows Server 2003, Windows Vista, Windows XP or Windows 2000 and ensure the latest Windows service pack is installed.
