IISAdminBase Registration

Description

The Internet Information Services (IIS) IISAdminBase must be registered on the computer running Microsoft® Baseline Security Analyzer, as well as on any remote computer being scanned, to perform the IIS-specific security checks. The IISAdminBase objects can be registered by installing the IIS Common Files that are included in the Windows® operating system. Note that a full installation of IIS is not required. To scan remote servers running IIS, only the IIS Common Files need to be installed on the computer running Microsoft Baseline Security Analyzer.
