Internet Explorer Security Zones

Issue

This checks the current and recommended security settings for the Internet Explorer zones for each local user on the scanned computer. Zones that are not using the recommended setting are flagged in the scan report.

Note

  • MBSA checks are based on the latest recommendations for a product or component. These recommendations may differ from the defaults that were in effect when a product or component was installed. Consequently, MBSA may report problems with Internet Explorer security zone settings even though no changes were made to the initial default settings.

Solution

Use the recommended Internet Explorer zone settings in the scan report.

Instructions

To select the recommended Internet Explorer zone settings

  1. Start Internet Explorer.
  2. On the Tools menu, click Internet Options.
  3. Click the Security tab, click each content zone, and then click Default Level to set the recommended security level. If Default Level is not enabled, you can re-enable it by changing the position of the security level slider.
