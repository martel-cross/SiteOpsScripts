Members of the Sysadmin Role

Check Description

This check determines the number of members of the Sysadmin role and displays the results in the security report.

Microsoft® SQL Server™ roles are used to group together logons that have the same rights to perform operations. The fixed server role, Sysadmin, gives system administrator rights to all of its members.

Note

  • If you get the "No permissions to access database" error message, you might not have permissions to the master database.
