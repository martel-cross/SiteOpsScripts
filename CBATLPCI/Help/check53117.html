Operating System Version

Check Description

This check determines which operating system and service pack is running on the scanned computer. Microsoft® Windows® Server 2008, Windows® Server 2003, Windows Vista, Windows XP, and Windows 2000 introduced new levels of reliability and availability for all of your business operations, such as improved control over file permissions.

To remain secure, Windows must have the latest publicly supported Windows operating system service pack installed.
